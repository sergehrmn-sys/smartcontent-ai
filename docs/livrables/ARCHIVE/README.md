## ⚠️ Pourquoi conserver les anciennes versions ?

- **Traçabilité du projet** : montre l'évolution du livrable au fil des sprints
- **Référence historique** : permet de retrouver une formulation antérieure si besoin
- **Sécurité** : évite la perte définitive d'un document si la version finale a un problème
